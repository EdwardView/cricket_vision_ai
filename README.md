Cricket Video Analysis Platform

Purpose :
This application is an AI-powered cricket video analysis tool that processes match footage to automatically detect, track, and classify key moments in cricket matches. It provides coaches, analysts, and cricket enthusiasts with detailed insights into player performance and match events.

Goals:

1.Automated Event Detection - Identify and classify critical match events including boundaries (FOUR), maximums (SIX), and dismissals (WICKET) with frame-accurate timestamps.
2.Ball Tracking & Trajectory Analysis - Track the cricket ball throughout each delivery, visualizing both the bowler's delivery path and the batsman's shot trajectory with real-time overlays.
3.Performance Metrics- Extract key statistics such as delivery speeds, shot types, scoring patterns, and wagon wheel visualizations to help understand batting tendencies.
4.Interactive Review - Enable users to scrub through video with synchronized overlays, jump to specific deliveries, and review individual ball events with detailed breakdowns.
5. Visual Analytics - Present data through intuitive visualizations including wagon wheels, shot breakdowns, and event timelines that make complex match data accessible at a glance.

Target Users:

 - Cricket coaches analyzing player technique
- Team analysts preparing opposition reports
- Broadcasters enhancing match coverage
- Cricket academies tracking player development
- Fans wanting deeper insights into match footage
